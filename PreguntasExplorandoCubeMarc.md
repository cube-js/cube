Este es un documento que tiene el contenido de las preguntas escritas de la practica: 



1. Frase sobre por qué has elegido ese proyecto



He elegido este proyecto porque me ha parecido interesante ya que es un framework de código abierto para construir aplicaciones web de análisis y paneles de control.



2\. Resumen del contenido del proyecto



Este es un proyecto que funciona como un servidor que maneja datos y analíticas. Su principal trabajo es traducir lo que pide una web a las consultas (SQL) necesarias para sacar datos de bases de datos grandes. El código está en JavaScript y TypeScript.



3\. Conclusión de lo que has aprendido



Aprendí que los proyectos grandes se organizan de forma compleja con un monorepo y usan TypeScript para mantener el código claro. He visto que el caching es muy importante para que las aplicaciones de datos sean rápidas, lo cual es clave al trabajar con grandes volúmenes de información.
