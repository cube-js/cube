# Okta

Cube Cloud supports authenticating users through [Okta][ext-okta],
which is useful when you want your users to access Cube Cloud using
single sign-on.

<InfoBox>

Available on [Enterprise and above plans](https://cube.dev/pricing).

</InfoBox>
