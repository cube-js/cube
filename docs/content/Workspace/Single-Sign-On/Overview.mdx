As an account administrator, you can manage how your team accesses Cube Cloud.
There are options to log in using email and password, a GitHub account, or
a Google account.

Cube Cloud also provides single sign-on (SSO) via identity provides supporting
industry-proven [SAML 2.0 protocol][wiki-saml], e.g., Okta, Google Workspace,
Azure AD, etc.

<SuccessBox>

Single sign-on is available in Cube Cloud on
[Enterprise](https://cube.dev/pricing) tier.
[Contact us](https://cube.dev/contact) for details.

</SuccessBox>
