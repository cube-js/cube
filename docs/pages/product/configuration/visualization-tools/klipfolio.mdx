## REST API

You can use the [REST API](/reference/rest-api) to connect Cube to Klipfolio PowerMetrics.

### Cube Cloud

When you have set up your Cube project in Cube Cloud, you need **REST API
endpoint** and **API Secret** for your setup in order to connect Klipfolio PowerMetrics
to Cube. Log in to your Cube Cloud account and follow the steps below.

**REST API endpoint** can be found on the&nbsp;<Btn>Overview</Btn> page:

<Screenshot src="https://ucarecdn.com/fddce1b2-fbdd-4711-82d9-3967af0e2641/"/>

**API Secret** can be found on the&nbsp;<Btn>Settings → Configuration</Btn>
page under&nbsp;<Btn>Environment variables</Btn> where it’s named
`CUBEJS_API_SECRET`:

<Screenshot src="https://ucarecdn.com/d978e95b-9e9c-443e-8b85-3106352a74cb/"/>

### Connecting in Klipfolio

Please see the [following instructions](https://support.klipfolio.com/hc/en-us/articles/23966341315095-PowerMetrics-Connecting-to-Cube-and-creating-metrics)
in the Klipfolio PowerMetrics Help Center.
