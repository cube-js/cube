# Sigma

[Sigma](https://www.sigmacomputing.com) is a business intelligence and analytics
platform with a spreadsheet-like interface.

## Connect from Cube Cloud

Navigate to the [Integrations](/product/workspace/integrations#connect-specific-tools)
page, click <Btn>Connect to Cube</Btn>, and choose <Btn>Sigma</Btn> to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Sigma using the [SQL API][ref-sql-api].

In Cube Core, the SQL API is disabled by default. Enable it and [configure
the credentials](/product/apis-integrations/sql-api#configuration) to
connect to Sigma.

## Connecting from Sigma

Sigma connects to Cube as to a Postgres database.


[ref-sql-api]: /product/apis-integrations/sql-api
