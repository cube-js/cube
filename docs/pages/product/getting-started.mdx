## Getting started with Cube Core

Cube Core is a headless, open-source semantic layer that powers the Cube Business Intelligence Platform. While Cube Core lacks frontend and AI features, you can use it to power your own embedded analytics or build your own AI agent or BI tool.

Cube Core is packaged and distributed as Docker images that can be run in a containerized environment. You can run Cube Core on your own infrastructure with Docker.
