# Source SQL

_Query your data sources directly in workbooks with AI assistance._

The Source SQL capability in Cube workbooks allows you to query your connected data sources directly, beyond just the semantic layer. Our AI agent can research your data source, help you build SQL queries, and refine results in real-time to provide deeper insights and exploration capabilities.
