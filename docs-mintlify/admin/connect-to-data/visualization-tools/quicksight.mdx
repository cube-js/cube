# Amazon QuickSight

[Amazon QuickSight](https://aws.amazon.com/quicksight/) is a cloud-based business
intelligence service provided by Amazon Web Services.

## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **Amazon QuickSight** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Amazon QuickSight using the [SQL API][ref-sql-api].

In Cube Core, the SQL API is disabled by default. Enable it and [configure
the credentials](/reference/sql-api#configuration) to
connect to Amazon QuickSight.

## Connecting from Amazon QuickSight

Amazon QuickSight connects to Cube as to a Postgres database.
