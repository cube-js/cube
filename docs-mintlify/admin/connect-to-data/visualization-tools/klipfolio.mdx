# Klipfolio PowerMetrics

[Klipfolio PowerMetrics](https://www.klipfolio.com/powermetrics) is a complementary
analytics solution that gives business users self-serve data access.

## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **Klipfolio PowerMetrics** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Klipfolio PowerMetrics using the [REST API](/reference/rest-api).

## Connecting in Klipfolio

Please see the [following instructions](https://support.klipfolio.com/hc/en-us/articles/23966341315095-PowerMetrics-Connecting-to-Cube-and-creating-metrics)
in the Klipfolio PowerMetrics Help Center.
