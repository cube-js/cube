# Hightouch

[Hightouch](https://hightouch.com) is a composable customer data platform (CDP).

## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **Hightouch** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Hightouch using the [SQL API][ref-sql-api].

In Cube Core, the SQL API is disabled by default. Enable it and [configure
the credentials](/reference/sql-api#configuration) to
connect to Hightouch.

## Connecting from Hightouch

Hightouch connects to Cube as to a Postgres database.
