# RudderStack

[RudderStack](https://www.rudderstack.com) is a warehouse-native customer data
platform (CDP).

## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **RudderStack** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to RudderStack using the [SQL API][ref-sql-api].

In Cube Core, the SQL API is disabled by default. Enable it and [configure
the credentials](/api-reference/sql-api#configuration) to
connect to RudderStack.

## Connecting from RudderStack

RudderStack connects to Cube as to a Postgres database.
