# Steep

[Steep](https://steep.app) is a modern analytics platform powered by and
specifically built for semantic layers. It features modern, user-friendly
UI as well as web, desktop, and mobile applications.

## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **Steep** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Steep using the [REST API](/api-reference/rest-api).

## Connecting from Steep

### Creating a data source

Copy **REST API endpoint** and **API Secret** and paste them into Steep form for
creating the Cube data source:

<Frame>
  <img src="https://ucarecdn.com/bde0cacf-9c25-4dce-af20-29855d20b3a0/" />
</Frame>

Hit&nbsp;**Continue** for your connection to be established and your
Cube metrics to be synced into Steep:

<Frame>
  <img src="https://ucarecdn.com/eecf1cc4-8912-44d8-a984-7808f540c66f/" />
</Frame>

### Defining and syncing metrics

See [Steep documention](https://help.steep.app/integrations/cube) for
guidance on defining and synchronizing metrics from Cube to Steep.
