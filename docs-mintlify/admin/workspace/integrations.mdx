# Integrations

Cube Cloud provides rich connectivity options for BIs, spreadsheets, and other
[visualization tools][ref-viz-tools], powered by [Semantic Layer Sync][ref-sls]
and [supported APIs][ref-apis]. Integrations suggest best connectivity options
for specific tools.

<Info>

Available on [all plans](https://cube.dev/pricing).

</Info>

You can [manage integrations](#manage-integrations), connect a [specific
tool](#connect-specific-tools), or [view API credentials](#view-api-credentials).

## Review integrations

You can review the statuses of APIs and enabled [Semantic Layer Sync][ref-sls]
integrations on the **Overview** page:

<Frame>
  <img src="https://ucarecdn.com/fabf25e5-c5b2-4117-9373-05768dcc21a9/" />
</Frame>

Click **Connect to Cube** to connect a [specific tool](#connect-specific-tools).
Click **API credentials** to [view credentials](#view-api-credentials)
for supported APIs.

## Manage integrations

Navigate to the **Integrations** page to manage all integrations.

<Frame>
  <img src="https://ucarecdn.com/4894508d-a178-4e47-a1b3-0d8c3c744f89/" />
</Frame>

Click **Connect to Cube** to connect a [specific tool](#connect-specific-tools).
Click **API credentials** to [view credentials](#view-api-credentials)
for supported APIs.
Manage [Semantic Layer Sync][ref-sls] integrations in the related page section.

## Connect specific tools

To connect a [visualization tool][ref-viz-tools], navigate to the **Integrations**
page and click **Connect to Cube**. Choose a specific tool or use the
search bar and the category filter in the sidebar to narrow the selection down.

<Frame>
  <img src="https://ucarecdn.com/949325be-e8e7-4f8f-98c6-3393a7db88ad/" />
</Frame>

Click on tools to get detailed instructions on connecting them to Cube Cloud: 

<Frame>
  <img src="https://ucarecdn.com/e881357d-3073-468b-975d-ddd4329d882c/" />
</Frame>

## View API credentials

To view [API credentials][ref-apis], navigate to the **Integrations** page
and click **API credentials**. Then, navigate to a tab that is dedicated
to a specific API.
