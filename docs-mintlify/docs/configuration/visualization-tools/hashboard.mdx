## Connect from Cube Cloud

Navigate to the [Integrations](/docs/workspace/integrations#connect-specific-tools)
page, click **Connect to Cube**, and choose **Hashboard** to get
detailed instructions.

## Connect from Cube Core

You can connect a Cube deployment to Hashboard using the [SQL API][ref-sql-api].

In Cube Core, the SQL API is disabled by default. Enable it and [configure
the credentials](/api-reference/sql-api#configuration) to
connect to Hashboard.

## Connecting from Hashboard

### Connecting a data source

To connect to Cube, click on &nbsp;**More → Data Sources** and
then click&nbsp;**+ Add Connection**:

<Frame>
  <img src="https://ucarecdn.com/1063b9bd-6133-4504-a576-008c383acadd/" />
</Frame>

Choose&nbsp;**PostgreSQL** and enter the connection credentials for Cube's SQL API:

<Frame>
  <img src="https://ucarecdn.com/2373ad16-9d3b-4353-8d5f-956cdddef3d6/" />
</Frame>

Next, choose your newly added data source and build the data model:

<Frame>
  <img src="https://ucarecdn.com/5851f4f8-feb4-4de0-88bc-f99ddb6b5365/" />
</Frame>

### Querying data

To query your data, go to the&nbsp;**Data models** page, choose your newly created
data model, and click&nbsp;**Explore**:

  <img src="https://ucarecdn.com/3eb1bc17-5ee7-4715-8be6-6fc17950c5e3/" />
</Frame>
