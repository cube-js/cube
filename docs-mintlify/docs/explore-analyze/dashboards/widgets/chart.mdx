---
title: Chart widget
description: Display a report from your workbook as a visualization on the dashboard.
---

Chart widgets display a report from the source [workbook][ref-workbooks] as a visualization on the dashboard. Every chart widget is backed by a tab in the workbook — when you publish a new version of the workbook, the chart widget on the dashboard updates accordingly.

## Adding a chart widget

In the [dashboard builder][ref-workbooks], open the **Charts** picker in the toolbar and select one or more reports from the workbook. A chart widget is added to the canvas for each selected report, inheriting the report's query, configured visualization, and styling.

If the picker is empty, create the report first in a workbook tab — only published reports are available to add to a dashboard.

## Chart types

You can use any of the built-in [chart types][ref-chart-types]:

- [**Bar**](/docs/explore-analyze/charts/chart-types/bar) — Compare values across categories with grouped, stacked, percentage, and horizontal variants.
- [**Line**](/docs/explore-analyze/charts/chart-types/line) — Show trends over time or any continuous dimension.
- [**Area**](/docs/explore-analyze/charts/chart-types/area) — Like line charts, but with the area below the line filled to emphasize magnitude or part-to-whole relationships.
- [**Scatter**](/docs/explore-analyze/charts/chart-types/scatter) — Plot two measures against each other to reveal correlation, clustering, and outliers.
- [**Pie & donut**](/docs/explore-analyze/charts/chart-types/pie) — Show how a single measure breaks down across a small number of categories.
- [**Heatmap**](/docs/explore-analyze/charts/chart-types/heatmap) — Visualize a measure across two dimensions using color intensity.
- [**Boxplot**](/docs/explore-analyze/charts/chart-types/boxplot) — Summarize the distribution of a measure across categories with quartiles and outliers.
- [**Table**](/docs/explore-analyze/charts/chart-types/table) — Show raw or aggregated rows with sortable columns and conditional formatting.
- [**KPI**](/docs/explore-analyze/charts/chart-types/kpi) — Highlight a single headline number, optionally with a comparison to a previous period.
- [**Map**](/docs/explore-analyze/charts/chart-types/map) — Plot geographic data on an interactive map.
- [**HTML**](/docs/explore-analyze/charts/chart-types/html) — Render a fully custom visualization using HTML and templating.

Configuration such as axes, color, series mapping, tooltips, and data labels is set on the report in the workbook and carries over to the dashboard.

## Interaction with filters and time grains

Chart widgets respect [filter widgets][ref-filter] and [time grain widgets][ref-time-grain] placed on the same dashboard. A single filter or time grain can drive multiple chart widgets at once — its value is applied to every chart whose query references the targeted dimension.

## Updating chart widgets

Chart widgets on a published dashboard reflect the most recent published version of the workbook. To change the query, switch the chart type, or restyle the visualization, edit the underlying report in the workbook and publish a new version of the dashboard.
