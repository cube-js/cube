---
title: Filter widget
description: Let dashboard viewers filter the data shown on the dashboard.
---

Filter widgets let dashboard viewers narrow down the data shown on the dashboard. A filter widget targets a dimension from your semantic model and applies the selected value to every chart widget on the dashboard whose query references that dimension.

## Adding a filter widget

In the [dashboard builder][ref-workbooks], click **Add Filter** in the toolbar. The new filter widget is added in an unconfigured state — click **Configure Filter** (or open the widget's settings menu) to pick a semantic view and a dimension.

## Operators by dimension type

The available operators depend on the type of the underlying dimension:

| Dimension type | Operators |
|---|---|
| **String** | `is`, `is not`, `contains`, `not contains`, `starts with`, `not starts with`, `ends with`, `not ends with`, `is null`, `is not null` |
| **Number** | `is`, `is not`, `greater than`, `greater than or equal`, `less than`, `less than or equal`, `is null`, `is not null` |
| **Time** | `is`, `is not`, `before date`, `before or on date`, `after date`, `after or on date`, `between`, `relative date`, `is null`, `is not null` |

## Single vs. multiple selection

Filter widgets can allow either a single value or multiple values. Configure this when adding or editing the filter — multi-select is the default for string dimensions, while time and number dimensions default to a single value.

## Default values

You can set a default value that's applied when the dashboard loads. Defaults are useful for scoping the dashboard to "this quarter" or "the user's region" without requiring viewers to interact with the filter first.

## Faceted filters

When multiple filter widgets target dimensions from the same semantic view, you can mark them as **faceted**. Faceted filters scope each other's value lists — selecting a value in one filter narrows the options shown in the others, so viewers only see combinations that exist in the data.

For example, on a sales dashboard with a **Country** filter and a **City** filter, marking both as faceted means selecting `United States` in the Country filter limits the City filter to U.S. cities only.

## Interaction with chart widgets

Filter widgets automatically apply to every [chart widget][ref-charts] on the dashboard whose query references the targeted dimension. Chart widgets that don't reference the dimension are unaffected, so a dashboard can mix scoped and unscoped views.
