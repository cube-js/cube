Widgets are the building blocks of a dashboard. Each tile placed on the canvas in the [dashboard builder][ref-workbooks] is a widget — a chart, a block of text, a control that viewers interact with, or an AI-generated summary. Combine them to assemble polished, interactive views of your data.

## Widget types

The dashboard builder supports five widget types:

- [Chart widget](/docs/explore-analyze/dashboards/widgets/chart) — Visualize a report from your workbook
- [Text widget](/docs/explore-analyze/dashboards/widgets/text) — Add titles, descriptions, and rich formatting in Markdown
- [Filter widget](/docs/explore-analyze/dashboards/widgets/filter) — Let viewers filter the data shown on the dashboard
- [Time Grain widget](/docs/explore-analyze/dashboards/widgets/time-grain) — Let viewers change the granularity of time-based dimensions
- [AI Widget](/docs/explore-analyze/dashboards/widgets/ai) — Generate narrative summaries of dashboard data on demand

In the dashboard builder, add widgets using the toolbar at the top of the canvas: pick reports from the **Charts** picker to add chart widgets, or click **Add Text**, **Add Filter**, **Add Time Grain**, or **Add AI Widget** for the other types.
