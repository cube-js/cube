Widgets are the building blocks of a dashboard. Each tile placed on the dashboard canvas is a widget — a chart, a block of text, a control that viewers interact with, or an AI-generated summary. Combine them to assemble polished, interactive views of your data.

## Widget types

Cube dashboards support four widget types:

- [Chart](/docs/explore-analyze/dashboards/widgets/chart) — Visualize a report from your workbook
- [Text / Markdown](/docs/explore-analyze/dashboards/widgets/text) — Add titles, descriptions, and rich formatting
- [Control](/docs/explore-analyze/dashboards/widgets/control) — Let viewers filter data or change how it's grouped
- [AI](/docs/explore-analyze/dashboards/widgets/ai) — Generate narrative summaries of dashboard data on demand

Add a widget by clicking **Add widget** in the dashboard editor and choosing the type you want.
